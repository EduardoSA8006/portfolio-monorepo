# Política de Segurança

## Versões suportadas

A branch `main` é a única versão atualmente suportada com correções de segurança.

## Reportando uma vulnerabilidade

Se você descobrir uma vulnerabilidade neste projeto, **não abra uma issue pública**.

Por favor, reporte de forma privada usando um dos canais abaixo:

- **GitHub Security Advisories** (preferencial): https://github.com/EduardoSA8006/portfolio-monorepo/security/advisories/new

Ao reportar, inclua:

- Descrição da vulnerabilidade e impacto potencial
- Passos para reproduzir
- Versão / commit afetado
- Qualquer prova de conceito (PoC), se aplicável

## Tempo de resposta

- Confirmação de recebimento: até 72 horas
- Avaliação inicial: até 7 dias
- Correção e divulgação coordenada: conforme severidade

Agradecemos a divulgação responsável.
